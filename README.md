## Available Scripts

In the project directory (Frontend or Backend), you can run:
### Installation 
### `npm install` 
### Startup
### `npm start`
### Testing
### `npm test`

## Used technologies

- React.js
- Redux
- Reactstrap (Bootstrap React library)
- React Testing Library
- Express.js
- MongoDB
- Mongoose

Note: "@types/jest" in package.json is for correct testing in JetBrains IDE only.
